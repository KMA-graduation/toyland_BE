# Khởi tạo database
- docker compose up -d 

# Chạy project
- Kiểm tra xem docker postgres đã chạy chưa
- Dùng node 18.17
- Chạy ```npm install```
- Chạy ```npm start:dev```

- Vào database xem có các bảng chưa
- Bật terminal mới chạy```npx ts-node seed.ts``` để tạo dữ liệu mẫu
